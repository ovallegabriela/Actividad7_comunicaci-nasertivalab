## Definición del problema

Hoy en día, muchos estudiantes tienen problemas para organizar sus tareas académicas. A veces tienen varias actividades al mismo tiempo, diferentes fechas de entrega y no saben por dónde empezar o incluso lo dejan todo a última hora porque creen que es fácil. Esto hace que se olviden de tareas, se estresen o entreguen trabajos tarde.

Además, muchas veces no llevan un control claro de qué ya hicieron y qué les falta, lo que afecta su rendimiento en los estudios y provoca que muchos pierdan sus cursos.
Por eso, se necesita una herramienta sencilla que ayude a los estudiantes a organizar mejor sus tareas, saber cuáles son más importantes y ver su progreso de una forma clara.

## Funcionalidades principales

### Registro de tareas

El usuario podrá agregar tareas ingresando información básica como el nombre, una pequeña descripción y la fecha de entrega.

### Asignación de prioridades

Cada tarea podrá marcarse como alta, media o baja prioridad, para que el estudiante sepa cuál debe hacer primero.

### Seguimiento del progreso

Las tareas podrán cambiar de estado, por ejemplo: pendiente, en proceso o completada, para llevar un control de lo que ya se hizo.

### Visualización de tareas

El sistema mostrará una lista organizada de todas las tareas, permitiendo verlas por fecha, prioridad o estado.

## Roles del sistema

### Usuario (estudiante)

El usuario es quien utiliza el sistema para organizar sus tareas académicas. Puede registrar tareas, asignar prioridades, actualizar el estado de las actividades y revisar su progreso.

### Administrador

El administrador se encarga de supervisar el sistema. Puede gestionar usuarios y asegurar que todo funcione correctamente, aunque este rol es opcional en una versión básica.

## Flujo de uso

1. El usuario ingresa al sistema.
2. Registra una nueva tarea ingresando el nombre, descripción y fecha de entrega.
3. Asigna una prioridad a la tarea (alta, media o baja).
4. La tarea se guarda y aparece en la lista.
5. El usuario puede ver sus tareas organizadas.
6. A medida que avanza, actualiza el estado de la tarea (pendiente, en proceso o completada).
7. El sistema muestra el progreso general del usuario.
